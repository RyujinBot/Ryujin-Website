Privacy Policy

1. Data Collection

Ryujin Editing Bot collects certain data for various functionalities, including but not limited to:

Channel IDs: Channel IDs are collected to enable the "Remove Background" function, which allows users to remove backgrounds from images within specific channels.

User IDs: User IDs are collected for the purpose of user profiles, favorites, and ratings. User profiles include social media links, links to created AMVs, and information about editing styles. Favorites include links to favorite AMVs.

Server IDs: Server IDs are used to enable the "Remove Background" function within specific Discord servers.

2. Use of Data

User Profiles: User profiles store social media links, links to created AMVs, and textual information about editing styles. This information is used to enhance user experiences and provide features such as user ratings and favorites.

Favorite AMVs: The bot stores links to favorite AMVs as provided by users. This feature allows users to access their favorite AMVs quickly.

Remove Background Function: Channel and server IDs are used exclusively for the "Remove Background" function, which serves to remove backgrounds from images as requested by users within specific channels.

3. Resources

All resources provided by Ryujin Editing Bot, including overlays, edit audios, project files, and presets, are sourced from freely available content on YouTube and other public websites. These resources are not leaked or obtained through unauthorized means. The bot aims to provide convenient access to these resources to enhance the creative process of its users.

4. Modifications to Privacy Policy

We reserve the right to modify this Privacy Policy at any time. Users are encouraged to review this policy regularly to stay informed about how their data is collected and used.

5. User Consent

By using Ryujin Editing Bot, users consent to the data collection practices outlined in this Privacy Policy.

If you have any questions or concerns about our data collection practices or this Privacy Policy, please feel free to contact us.
